# TestAppStore

## Get started

### Clone the repo

```sh
git clone https://github.com/mkuziv/test-app.git
cd test-app-store
```

### Install npm packages

```sh
npm install 
ng serve
```


### Build

Run 
```sh
ng build 
```
to build the project. The build artifacts will be stored in the dist/ directory. Use the `--prod` flag for a production build.
